Detailed Description

This class implements the encoder a DC motor to the ME 405 board.

Example:

1 ''' Create an Encoder object '''
2 enc = encoder.Encoder(pyb.Pin.board.PB6, pyb.Pin.board.PB7,4)
3 ''' Get the current location of the motor '''
4 location = enc.read()
5 ''' Zero the encoder reading at the current location '''
6 enc.zero()
