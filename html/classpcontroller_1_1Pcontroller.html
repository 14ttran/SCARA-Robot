Detailed Description

This class implements a proportional controller for the motor.

It is intended to be called continuously in order to properly control the location of the motor

Example of how to use each function individually:

1 ''' Create an Pcontroller object '''
2 motorvoltage = 12 #Volts
3 controller = pcontroller.Pcontroller(motorvoltage)
4 ''' Set the destination of the motor '''
5 destination = 1000
6 controller.setsetpoint(destination)
7 ''' Set the gain and saturation values of the controller '''
8 kp = 0.01
9 ki = 0
10 psat = 4000
11 nsat = -4000
12 controller.setgain(kp,ki,psat,nsat)
13 ''' Calculate the pwm value to send to the motor '''
14 current_location = 0
15 pwm = controller.control(current_location)
