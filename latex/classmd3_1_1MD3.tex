\subsection{Detailed Description}
This class implements a motor driver for the ME 405 board. 

Its function is identical to the Motor\+Driver class, but contains differences due to the way the physical motor driver generates pwm signals.

Example\+: 
\begin{DoxyCode}
1 \textcolor{stringliteral}{''' Create a MotorDriver object '''}
2 md = \hyperlink{classmd3_1_1MD3}{md3.MD3}(pyb.Pin.board.PB0, pyb.Pin.board.PB8, pyb.Pin.board.PB9, 3)
3 \textcolor{stringliteral}{''' Drive the motor with a pwm signal '''}
4 pwm = 50 \textcolor{comment}{#set a pwm of 50%    }
5 md.set\_duty\_cycle(pwm)
\end{DoxyCode}
